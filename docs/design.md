# 页面视觉原型 (UX Design)

## 视觉基调
Notion 风格（背景采用 #FAFAFA 象牙白，大量留白，图标使用简洁的 Line Icon，强调信息层级而非花哨装饰）。视觉风格为Notion 风格，要求程序页面背景为白色偏黄的阅读模式。

## 页面设计

| 页面名称 | 视觉核心 | 关键交互 |
| --- | --- | --- |
| 首页 (Home) | 活跃学生卡片阵列（GridView 宫格视图） | 长按头像进入编辑，单击多选，一键全选 |
| 反馈页 (Process) | 分屏设计：上选人，下写内容 | 搜索框支持拼音简码，快速添加作业，记忆功能自动载入昨日作业项 |
| 统计页 (Dashboard) | 环形图 + 预警列表 | 点击预警用户，可直接拨打电话或复制微信名，点击"欠费/临期"学员可一键复制家长微信号 |
| 设置页 (Settings) | 列表式功能菜单 | 作业库管理、反馈图模版更换、数据导出备份 |

## 页面功能说明

### 首页 (Home)
- 采用 GridView（宫格视图） 展示所有"在读"学员
- 交互：点击头像多选（或一键全选），点击底部大按钮 「完成今日登记」
- 系统动作：批量创建今日待反馈记录，并扣除相应课时

### 反馈页 (Process)
- 顶部导航：横向滚动展示今日已签到的学生头像
- 记忆功能：点击学生头像，下方内容区自动载入该学生昨日的作业项（减少输入量）
- 任务编辑：点击 + 号弹出作业库，支持搜索，添加到当前反馈里
- 支持手动修改每项状态（已完成、需加强、未带书）

### 统计页 (Dashboard)
- 显示月度/周度消课总数趋势图
- 阈值预警：当剩余课时 ≤ 3次时，给予橙色高亮提示

### 设置页 (Settings)
- 作业库管理：支持首字母拼音快捷搜索（如输入 PY 匹配 拼音）
- 反馈图模版更换：预设反馈图的视觉风格（简约、可爱、专业）
- 数据导出备份功能
